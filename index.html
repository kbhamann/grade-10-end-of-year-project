GAME

Game Description and Instructions

This is my attempt at semi-complex movement in a platformer. There is no real goal to this game, although if I had more time and the opportunity to discover different ways to code, I would add enemies and procedural generation so that you would move around the map by ramming into enemies to kill them.

Ball Movement

Right now, you can use WASD to move a ball around. If you hit a wall while pressing the button that leads towards the wall, you will wall run or climb the wall. If you don't hit the button, you will bounce off of the wall. This is the most complex part of the code. If you use the mouse in non-build mode (no text at top saying build mode), you can teleport the ball around the map to gain height it is on a two second cool down.

Build Mode

There are two modes for the mouse. Build mode can be toggled using space bar. In build mode, you can build platforms like a map editor. The way that build mode works is that you click on two different points to make a structure. Then, the collisions only work if the first click is below the second. Otherwise the if statements don't work.
